## 游戏简介

扮演自定义女主角，在回合制行程安排中与Kpop爱豆发展关系。经历**攻略→恋爱→结婚**三个阶段，面对查岗、D社偷拍、公司干涉等现实挑战，走向属于你的独特结局。

## 特性

- 🎭 **完全自定义**：女主和爱豆的姓名、性别、身份、性格全部自由设定
- 📱 **真实手机体验**：聊天、SNS、新闻、通话——游戏内的手机就像真的一样
- 🤖 **AI驱动对话**：通过DeepSeek API，爱豆会根据自己的性格和你当前的关系动态回复
- 🔀 **15种结局**：没有"正确"路线，每个选择都是你的故事
- 🌓 **动态视觉**：色调随你的心情和处境变化
- 📸 **自定义照片**：上传爱豆头像和手机壁纸

## 技术

- 纯 HTML/CSS/JS（零依赖）
- DeepSeek API 驱动动态内容
- 数据全本地存储（隐私安全）
- 部署在 GitHub Pages

## 系统要求

| 平台 | 最低版本 |
|------|---------|
| Safari (iOS) | 14.1+ |
| Chrome (Android) | 90+ |
| Samsung Internet | 14+ |
| 其他 | 2021年后发布的现代移动浏览器即可 |

> ⚠️ 旧版iOS Safari（<14.1）不支持CSS flexbox `gap`属性，布局可能错乱。建议将系统更新至iOS 14.5以上。

## 隐私说明

🔒 **所有数据仅存储在你的设备上**，不会有任何数据上传到服务器。游戏使用浏览器的 localStorage 和 IndexedDB 存储：

- 游戏存档 → 浏览器 localStorage（3个存档槽）
- 照片/头像 → 浏览器 IndexedDB
- 聊天记录 → 浏览器 IndexedDB
- API Key → 浏览器 localStorage（base64编码）

你可以随时通过浏览器设置清除这些数据。
