# Kpop嫂子模拟器 — 开发执行步骤

> 版本：v1.0 | 最后更新：2026-06-06

---

## 开发原则

1. **每阶段独立可验收** — 不做完前一阶段，不进下一阶段
2. **每个阶段结束 → 更新开发日志** — 记录完成项和待办项
3. **遇到技术阻碍 → 记录到日志，降级方案备好** — 不等不卡
4. **先跑通再优化** — 能点能用比好看重要
5. **中文注释** — 所有代码中文注释，方便以后理解
6. **Git 提交** — 每个阶段至少一次 commit
